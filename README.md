# Finance Signal Lab

**Finance Signal Lab** is my personal space for exploring markets, finance, and the signals shaping today’s economic landscape. Shaped through usual channels of knowledge including generic news updates, industry experts and self reserached topics.  
This repository powers my GitHub Pages blog, where I share opinion driven takes on macro trends, quantitative ideas, risk frameworks, and anything else that catches my attention in the world, that has direct or indirect relation to the field of finance.

The goal is simple:  
Create a clean, honest, and accessible place to write about what I’m learning, what I’m questioning, and how I interpret the current financial environment.

## 🧭 How This Site Works

This repository uses GitHub Pages + Jekyll to generate a simple, dark‑mode blog.

- All posts live in the `_posts/` folder  
- Each post follows the format:  
  `YYYY-MM-DD-title.md`
- The site automatically builds whenever I push a new post  
- The homepage provides an overview of what this blog is about  
- New posts appear as standalone pages

---

## ✍️ How to Read New Posts

Since the theme is intentionally minimal:

- Visit the live site  
- Scroll through the homepage  
- Click any post link I add there  
- Or navigate directly using the post URL format:  
  `/YEAR/MONTH/DAY/post-title.html`

Example:  
`/2026/03/16/helium-contest.html`
